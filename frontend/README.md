# IDBI MSME Financial Health Card — Frontend

Bank lending underwriting UI for the **IDBI MSME Financial Health Card**, built by
**Team Anvay**.

React + Vite + TypeScript + Tailwind CSS. Proxies `/api` to the Express backend on port 4000.

```bash
npm install
npm run dev    # http://localhost:5173
```

See the root [README.md](../README.md) for full-stack architecture, API reference, and setup.
